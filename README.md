# TV Repair Assistant (Offline‑ready)

A PWA for TV repair diagnosis. Works entirely offline using a cached database.

## Files
- `index.html` – main app (add your own)
- `offline.html` – custom offline page
- `manifest.json` – PWA manifest
- `sw.js` – service worker with offline fallback

## Usage
1. Clone the repo.
2. Serve the folder with any static server (e.g., `npx serve .`).
3. Open in a browser, then go offline to see the custom offline page.

## Deploy to GitHub Pages
1. Push to a `gh-pages` branch or set the main branch as source.
2. Enable GitHub Pages in repository settings.
